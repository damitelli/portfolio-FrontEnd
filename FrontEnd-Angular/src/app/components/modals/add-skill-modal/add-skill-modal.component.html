Agregar habilidad

El nombre es requerido (mínimo 1 caracter y máximo 50 caracteres).

El nombre debe ser de 1 o más caracteres.

El nombre debe tener como máximo 50 caracteres.

El porcetaje de dominio es requerido (mínimo 1% y máximo 100%).

El porcetaje de dominio debe ser como mínimo de 1%.

El porcetaje de dominio debe ser como máximo de 100%.

El porcetaje de dominio solo acepta números entre 1 y 100.

El tipo de habilidad es requerido.
